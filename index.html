Mes Projets

ORINOCO

Site E-commerce réalisé dans le cadre de ma formation chz OpenClassRoom. Le projet Consite a créer le coté frontend du site e-commerce. Faire des requêtes asynchrones vers une API et construire le site a partir des donnée reçus de cette API. Le site est divisé en 4 pages: Accueil, Produit, Panier, Commande et Confirmation.

GROUPOMANIA

Un reseau social pour l'entreprise groupomania, sous forme d'application WEB. Le site est développé pour le mobile first. Afin de valider le parcours de développeur web chez OpenClassRoom. En backend j'ai utilisé Nodejs avec le module Express, Mysql comme base de donnée et SQL pour les CRUD. En frontend j'ai préféré utiliser React. L'application est trés fluid, aucub=n chargement sauf pour les requêtes. Les employés peuvent s'inscrire poster des medias (images, gif), commenter liker et se desinscrire. l'application respect la RGPB et l'OWASP pour la securité et la gestion des données utilisateur.

Voir le repos github

CAT-AMAZON

Site de gestion des adherents de la CAT et salaries chez Amazon. Ce site a été développé avec PHP native, en uilisant la POO programmation oriented object. Connexion sécurisé, les données sont sur une base de donnée MYSQL. Différents options de création d'adhérent, modification des données, ajout de cotisation filter et exporter les données. Le tous en respectant les recommandations de la RGBD.
